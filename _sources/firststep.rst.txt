Авторизация
===========

При запуске приложения вас встречает окно авторизации. В нем содержится два поля: vk_token (`Ключ доступа Вконтакте`_.); vk_group_id (`Уникальный идентификатор сообщества`_.). Для выполнения входа в приложение необходимо заполнить оба поля на странице и нажать кнопку "Log in". Описание каждого представлено ниже. Данные сохраняются после первой авторизации.

Ключ доступа Вконтакте
----------------------

Первым полем на странице авторизации является ключ доступа (access_token).

Токен необходим, прежде всего, для связи сервиса (в данном случае - ВК товаров) со сторонним приложением (в данном случае - МаркетБот). 

Алгоритм получения
~~~~~~~~~~~~~~~~~~

    #. Перейти по ссылке_.
    #. Нажать "Разрешить"
    #. Скопировать адресную строку и вставить в МаркетБот, в поле vk_token

.. _ссылке: https://oauth.vk.com/oauth/authorize?client_id=6121396&scope=1073737727&redirect_uri=https://oauth.vk.com/blank.html&display=page&response_type=token&revoke=1&slogin_h=ab7f11e9b4a8b55f54.322104d60e6366c1d3&__q_hash=cc99b3297d940b1f788a39428a48e232

* *Токен выпускается один раз, перевыпуск при каждой новой сессии не требуется*
* *Чтобы выпустить ключ доступа, вы должны быть администратором сообщества*

Уникальный идентификатор сообщества
----------------------

Вторым полем на странице авторизации является уникальный идентификатор сообщества (groupId). Используется для определения сообщества, с которым будет взаимодействовать приложение. 

У любого сообщества может быть два уникальных идентификатора - числовой и буквенный (латиницей). МаркетБот может работать с любым из них.

Как получить идентификатор своего сообщества вы можете посмотреть здесь_. Также, в качестве идентификатора может использоваться ссылка на сообщество.



.. _здесь: https://vk.com/faq18062
