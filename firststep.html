Авторизация¶

При запуске приложения вас встречает окно авторизации. В нем содержится два поля: vk_token (Ключ доступа Вконтакте.); vk_group_id (Уникальный идентификатор сообщества.). Для выполнения входа в приложение необходимо заполнить оба поля на странице и нажать кнопку «Log in». Описание каждого представлено ниже. Данные сохраняются после первой авторизации.

Ключ доступа Вконтакте¶

Первым полем на странице авторизации является ключ доступа (access_token).

Токен необходим, прежде всего, для связи сервиса (в данном случае - ВК товаров) со сторонним приложением (в данном случае - МаркетБот).

Алгоритм получения¶

  1. Перейти по ссылке.

  2. Нажать «Разрешить»

  3. Скопировать адресную строку и вставить в МаркетБот, в поле vk_token

  • Токен выпускается один раз, перевыпуск при каждой новой сессии не требуется

  • Чтобы выпустить ключ доступа, вы должны быть администратором сообщества

Уникальный идентификатор сообщества¶

Вторым полем на странице авторизации является уникальный идентификатор сообщества (groupId). Используется для определения сообщества, с которым будет взаимодействовать приложение.

У любого сообщества может быть два уникальных идентификатора - числовой и буквенный (латиницей). МаркетБот может работать с любым из них.

Как получить идентификатор своего сообщества вы можете посмотреть здесь. Также, в качестве идентификатора может использоваться ссылка на сообщество.
